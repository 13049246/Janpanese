#### 1.小句简体 + 時
| 简体  | 時 | 意味 |
| :----: | :----:  | :----: |
| ～する  | 時 | 动作实现前前项还未完成 |
| ～した  | 時 | 动作实现前前项已完成 |
| 形一    | 時 | 前项是后项动作进行时间 |
| 形二  | な時 | 前项是后项动作进行时间 |
| 名    | の時 | 前项是后项动作进行时间 |

* 表示一次性、个别的情况是，用"～時に"




#### 2.动 + ながら
* 同一主体进行两个动作，后面为主要动作




#### 3.小句（简体）でしょう？
* | 升调 | 确认对方比自己更熟悉的事 |
* | 降调 | 意见不同，叮咛对方 |



#### 4.动ています
* 动作正在进行
* 动作的结果、状态
* 反复 习惯



#### 5.名で
*　表示原因、理由



#### 6.と／に　会います
* に　表示对象
* と  表示互相动作
